Class Descriptions

All classes are 60 minutes with the exception of BARRE EXPRESS.

Barre Basic

For newbies to barre. Full body workout. Class incorporates using your own body weight as resistance combined with small hand weights and stability balls which overload the muscle. Each class ends with several stretch tracks to assist with that longer looking graceful physique.

Learn More

Barre Attack

For those experienced in barre with higher intensity and a faster pace of class. Sweat burn laugh!

Learn More

Barre Express

45 minute lunchtime session tailored to the class. Combination of Barre Basic and Barre Attack. Suitable for all levels.

Learn More

Barre Classic

Suitable for more mature ages or those recovering from surgery or injuries. The most gentle low intensity.
